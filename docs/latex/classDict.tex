\doxysubsection{Detailed Description}
A class contining information loaded from a configuration file. 

This class creates a set of unordered maps which translate a {\ttfamily std\+::string} key into {\ttfamily double}, {\ttfamily int}, or {\ttfamily std\+::string} values read from an input configuration file. ~\newline
